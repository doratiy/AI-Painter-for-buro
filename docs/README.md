# Рисовалка с ИИ | Крокодил

## Описание сайта
На сайте "Рисовалка с ИИ | Крокодил" можно играть в крокодил с искуственным интелектом. Вы зарабатываете очки, когда угадываете, что нарисовал ИИ и когда ИИ не смог отгадать ваш рисунок.

## Примеры запуска оффлайн версии сайта
Чтобы оффлайн запустить версию сайта надо - открыть репозиторий у себя на компьютере, открыть папку ```docs``` и найти файл ```index.html```, дальше выбираете с помощью какого браузера его окрыть, после чего у вас запустится ```localhost``` и вы сможете пользоваться сайтом.

## Где найти сайт в интернете
Чтобы перейти на мой сайтб, вам надо в браузере перейти по ссылке [Рисовалка с ИИ | Крокодил](https://doratiy.github.io/AI-Painter-for-buro/)

## Идеи для улучшения сайта
Стоит заменить упрощение рисунка от ИИ - вместо текста заставить его рисовать. Можно добавить вызов ML5.js или OpenAI для угадывания.
